## Git Preview

### Description of Git
> Version Control System
- 파일의 이름을 변경하지 않고 파일의 버전을 관리할 수 있다는 것이  git 의 맹점
- Backup : 내 코드 백업 가능
- Recovery : 코드를 이전으로 돌릴 수 있음
- Collaboration : 협업 하는데 편리
- example) CVS, SVN, GIT

#### Git is incredibly **complex**

### Make a repository
~~~
> pwd : 현재 위치를 알려줌

> mkdir : 폴더 생성

> ls -al : 현재 디렉토리의 파일명을 보여줌

> git init : 현재 디렉토리를 git의 (버전) 저장소로 만듦
~~~~
